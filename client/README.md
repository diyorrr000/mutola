# Mutolaa Platformasi

Ushbu loyiha o'zbek adabiyoti va dunyo durdonalarini o'qish uchun mo'ljallangan platforma.

## Loyiha tarkibi:

- **/client**: React (Vite) yordamida qurilgan foydalanuvchi interfeysi (Frontend).
- **/backend**: Node.js va Express yordamida qurilgan API xizmati (Backend).

## Mahalliy ishga tushirish:

### Backend:
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend:
1. `cd client`
2. `npm install`
3. `npm run dev`

## Deploy:
- **Frontend**: Vercel
- **Backend**: Render
